## 与官方文档的差异

MiMo 官方文档描述：思考模式 Agent 对话中存在工具调用时，后续请求必须完整回传
相关历史 `reasoning_content`，否则返回 400。

本次真实 API 探针未复现该行为。当前不能据此得出“无需保存 `reasoning_content`”
的结论，原因可能包括：

- 当前端点或模型行为已变化，但文档尚未更新；
- 仅特定消息位置或特定请求形态触发校验；
- 省略字段、空字符串与截断字段的行为不同；
- 当前测试账号、区域或 API 路由使用了不同兼容策略。

## 决策影响

- Mimodex 仍必须完整保存并回传 `reasoning_content`。
- 将此项标记为“文档与实测不一致”，不得标记为通过。
- 后续矩阵已完成：两个模型在省略、空字符串和截断三种模式下均接受请求。
