# 验证证据目录

本目录保存按日期形成的 MiMo Provider、Runtime 和桌面行为验证证据。证据文档记录实验
发生时的模型、提交、命令、结果和已知限制，属于历史事实，不随项目当前进度改写。

需要了解当前项目阶段、最新 Windows Preview 和剩余工作时，请阅读：

- [Mimodex 当前项目状态](../../CURRENT_STATUS.md)
- [MiMo Provider Spike 当前状态](../MIMO_PROVIDER_SPIKE_STATUS.md)
- [Windows 11 私测验收清单](../WINDOWS_11_PRIVATE_BETA_ACCEPTANCE.md)

新增证据时应：

- 使用 `YYYY-MM-DD-主题.md` 命名；
- 记录精确模型、提交和运行链接；
- 不包含 API Key、私有项目内容或未脱敏原始请求；
- 明确区分实测事实、推断和尚未验证事项。
